# Gelismis PHP Haber Sistemi

Modern, hizli ve SEO uyumlu haber portali scripti. Tamamen PHP ve PDO kullanilarak gelistirilmistir[cite: 2].

## Ozellikler

* Dinamik icerik yonetimi ve sinirsiz kategori destegi[cite: 3].
* Kullanici kayit, giris ve profil yonetimi altyapisi[cite: 2, 7].
* Kullanicilar icin haberleri favorilere ekleme ve sonraya kaydetme islevleri[cite: 2, 6, 7].
* Ana sayfada anlik toplam haber, okunma ve gunluk icerik istatistikleri[cite: 5].
* Son dakika haberleri icin ozel animasyonlu kayan bulten alani[cite: 4].
* Haberler icin detayli yorum sistemi ve onay mekanizmasi[cite: 9].
* Sitenin farkli alanlari icin reklam yonetimi[cite: 9].
* Yonetici islem loglari ve sunucu ici medya yonetimi[cite: 9].
* .htaccess yapilandirmasi ile tam SEO uyumlu temiz URL yapisi[cite: 8].
* Dis servislerden (hava durumu vb.) veri cekebilmek icin cURL tabanli API entegrasyonu[cite: 2].
* Vanilla JS ve CSS3 kullanilarak hazirlanmis, tamamen mobil uyumlu responsive tasarim[cite: 4].
* Dinamik klasor tespiti yapan esnek konfigirasyon yapisi[cite: 10].

## Kurulum

1. Tum dosyalari sunucunuzun ana dizinine veya bir alt klasore yukleyin.
2. Veritabani yapisini olusturmak icin sirasiyla `haber.sql`, `users.sql` ve `admin.sql` dosyalarini veritabaniniza ice aktarin[cite: 3, 7, 9].
3. `config.php` dosyasini acarak kendi sunucunuza uygun veritabani (DB_HOST, DB_NAME, DB_USER, DB_PASS) bilgilerinizi tanimlayin[cite: 10].
4. Sistemin calismasi icin gerekli kurulum tamamlanmistir.
